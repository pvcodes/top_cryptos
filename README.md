<h1 align=center> Top Crypto</h1>
<p>
A MEN stack webapp, which fetches the crypto data from wazirX API


</p>

### STACK USED:
    
    - Nodejs
    - Expressjs
    - Mongoose (for DB)
    - Warix API


## Hosting the app locally:



- Clone this repo using `git clone`
- cd into the root folder.



- Install the `node modules` via `npm install` 

- then run, `node index.js`
